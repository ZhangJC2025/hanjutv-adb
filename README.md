### Introduction
This is a UserScript for removing ads from [hanju1.tv](hanju1.tv). It supports both desktop and mobile devices.

For desktop, we recommend installing using [Tampermonkey](https://www.tampermonkey.net); for Android, we recommend installing using the Via browser; and for iOS, we recommend using [Userscripts](https://apps.apple.com/us/app/userscripts/id1463298887).
